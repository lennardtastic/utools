About this converter

SVG stands for Scalable Vector Graphics. Many organizations nowadays use SVG files on their website for the logos. SVG is used to define so-called vector-based graphics for the Web. SVG defines these graphics in an XML format which allows graphics do NOT lose any quality if they are zoomed or resized. In addition, every element and every attribute in SVG files can be animated. While SVG image brings many benefits they are hard to display in older browsers and they do not always render as you would expect. For this reason, you may want to use convert your SVG files to PNG images. Drop any SVG file into the drop area to have it converted to a PNG image. The conversion fully happens on the client side. After you've downloaded your png image we recommend that you go to Tiny PNG to optimize the PNG image for achieving the best overall performance.
